# Self-balancing robot with ESP32 WROOM

## Pinout

### LED Wiring

- RED: GPIO32
- GREEN: GPIO33
- BLUE: GPIO25

### MPU9250 Wiring

- SCL: GPIO22
- SDA: GPIO21
- INT: GPIO19

### Motor 1 Wiring

- EN: GPIO15
- DIR: GPIO14
- STEP: GPIO12

### Motor 2 Wiring

- EN: GPIO16
- DIR: GPIO26
- STEP: GPIO27
